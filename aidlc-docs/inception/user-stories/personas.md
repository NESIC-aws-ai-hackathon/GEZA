## ペルソナ 1：田中 健太（Kenta）— 考えたくないけど失敗したくない人

| 項目 | 内容 |
|------|------|
| 年齢 | 28歳 |
| 職業 | IT企業 SE（3年目） |
| 役職 | 一般社員 |
| 利用目的 | やらかしたことを入れるだけで、謝罪台本から手土産まで全部AIに作ってほしい |
| 典型シナリオ | 顧客への納期遅延、上司への報告ミス |
| 課題 | 謝罪の場で頭が真っ白になる。自分で文面を考える余裕がない |
| 技術リテラシー | 高（スマホ・PCに慣れている） |
| 利用頻度 | 月1〜2回（実案件発生時） |
| 期待する価値 | **入力から台本完成まで1分以内**。自分で文面を一字も書かずに謝罪完了 |
| 感情状態 | 焦り・不安を抱えた状態でアプリを開く。短時間で成果がほしい |

**主な利用フロー**: トップ画面 → 一言入力 → 角度アセスメント → 台本フル生成 → （任意）リハーサル → 頭を下げる

**関連 Epic**: Epic 1, 2, 4, 5, 6

## ペルソナ 2：佐藤 美咲（Misaki）— 謝罪が苦痛じゃなくゲーム感覚で終わってほしい人

| 項目 | 内容 |
|------|------|
| 年齢 | 24歳 |
| 職業 | 広告代理店 アカウントプランナー（1年目） |
| 役職 | 一般社員（新人） |
| 利用目的 | 謝罪を自分で考えず、AIが全部作ってくれるのをゲーム感覚で楽しみたい |
| 典型シナリオ | ストーリーモードで謝罪ボスに挑戦。失敗してもAIが台本を書き直してくれるので反省不要 |
| 課題 | 社会人経験が浅く、謝罪の「型」を知らない。クレーム対応が怖い |
| 技術リテラシー | 高（ソシャゲに慣れている） |
| 利用頻度 | 週2〜3回（通勤中・休憩中） |
| 期待する価値 | **AIが全部作るので自分は考えなくていい**。失敗してもAIが対応してくれる安心感 |
| 感情状態 | カジュアルにアプリを開く。楽しみながら学びたい |

**主な利用フロー**: トップ画面 → ストーリーモード → 難易度・テーマ選択 → AI製台本で謝罪ボス戦 → リハーサル → カルテで確認

**関連 Epic**: Epic 1, 3, 4, 6

## ペルソナ 3：山田 誠一（Seiichi）— 謝罪も指導もAIに任せて自分は判断だけしたい上司

| 項目 | 内容 |
|------|------|
| 年齢 | 42歳 |
| 職業 | メーカー 課長 |
| 役職 | 管理職（部下10名） |
| 利用目的 | 謝罪文面も指導の言葉もAIが全部作ってくれる。自分はGO/NO-GOの判断だけしたい |
| 典型シナリオ | 部下への注意指導、自身が上位管理職へ謝罪する場面 |
| 課題 | 感情的になって人格否定してしまう。「なぜできないんだ」と言ってしまう |
| 技術リテラシー | 中（業務用PCとスマホは使えるが、ゲームには不慣れ） |
| 利用頻度 | 月2〜3回（部下の問題発生時 + 自身の謝罪時） |
| 期待する価値 | **自分で文面を書かなくてOK**。パワハラリスク「低」定着 |
| 感情状態 | イライラ・困惑した状態でアプリを開く。すぐに使えるシンプルさを求める |

**主な利用フロー**:
- **指導練習**: トップ画面 → 上司向けフィードバック → 指導内容入力 → AI検知・改善案
- **自身の謝罪**: トップ画面 → 一言入力 → 角度アセスメント → 台本フル生成 → （任意）リハーサル

**関連 Epic**: Epic 1, 2, 4, 5, 6, 7
