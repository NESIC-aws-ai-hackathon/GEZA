## ペルソナ 1：田中 健太（Kenta）— 本番前に練習したい人

| 項目 | 内容 |
|------|------|
| 年齢 | 28歳 |
| 職業 | IT企業 SE（3年目） |
| 役職 | 一般社員 |
| 利用目的 | 実際にやらかした案件の謝罪を、本番前にリハーサルしたい |
| 典型シナリオ | 顧客への納期遅延の謝罪、上司への報告ミスの謝罪 |
| 課題 | 謝罪の場で頭が真っ白になる。言ってはいけないことを言ってしまう |
| 技術リテラシー | 高（スマホ・PCに慣れている） |
| 利用頻度 | 月1〜2回（実案件発生時） |
| 期待する価値 | 本番でしくじらないための準備。NGワードの事前把握。再発防止策のテンプレート |
| 感情状態 | 焦り・不安を抱えた状態でアプリを開く。短時間で成果がほしい |

**主な利用フロー**: トップ画面 → 実案件モード → やらかし入力 → 謝罪相手生成 → 謝罪練習 → 謝罪後支援 → カルテ保存

**関連 Epic**: Epic 1, 2, 4, 5, 6

## ペルソナ 2：佐藤 美咲（Misaki）— ゲーム感覚で謝罪力を鍛えたい人

| 項目 | 内容 |
|------|------|
| 年齢 | 24歳 |
| 職業 | 広告代理店 アカウントプランナー（1年目） |
| 役職 | 一般社員（新人） |
| 利用目的 | 謝罪スキルをゲーム感覚で身につけたい。暇な時間に練習したい |
| 典型シナリオ | ストーリーモードで謝罪ボスに挑戦。難易度を上げて腕試し |
| 課題 | 社会人経験が浅く、謝罪の「型」を知らない。クレーム対応が怖い |
| 技術リテラシー | 高（ソシャゲに慣れている） |
| 利用頻度 | 週2〜3回（通勤中・休憩中） |
| 期待する価値 | ゲーム的な達成感。謝罪パターンの学習。自分の弱点の可視化 |
| 感情状態 | カジュアルにアプリを開く。楽しみながら学びたい |

**主な利用フロー**: トップ画面 → ストーリーモード → 難易度・テーマ選択 → 謝罪ボス戦 → 謝罪練習 → カルテで成長確認

**関連 Epic**: Epic 1, 3, 4, 6

## ペルソナ 3：山田 誠一（Seiichi）— 部下を指導したい上司

| 項目 | 内容 |
|------|------|
| 年齢 | 42歳 |
| 職業 | メーカー 課長 |
| 役職 | 管理職（部下10名） |
| 利用目的 | 部下への注意・指導を、パワハラにならない形で練習したい。自分自身の謝罪練習にも使う |
| 典型シナリオ | 部下のミスへの注意指導。自身が上位管理職への謝罪を練習 |
| 課題 | 感情的になって人格否定してしまう。「なぜできないんだ」と言ってしまう |
| 技術リテラシー | 中（業務用PCとスマホは使えるが、ゲームには不慣れ） |
| 利用頻度 | 月2〜3回（部下の問題発生時 + 自身の謝罪時） |
| 期待する価値 | パワハラ回避の言い換え表現。建設的なフィードバック手法。自身の謝罪力向上 |
| 感情状態 | イライラ・困惑した状態でアプリを開く。すぐに使えるシンプルさを求める |

**主な利用フロー**:
- **指導練習**: トップ画面 → 上司向けフィードバック → 指導内容入力 → AI検知・改善案
- **自身の謝罪**: トップ画面 → 実案件モード → やらかし入力 → 謝罪練習 → 謝罪後支援

**関連 Epic**: Epic 1, 2, 4, 5, 6, 7
