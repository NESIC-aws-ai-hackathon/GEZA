## 指示
上記の謝罪文に対して、相手プロフィールの性格・感情状態を踏まえて反応してください。
謝罪の内容を評価し、以下のJSON形式のみで回答してください（説明文や前置きは不要）。

感情ラベルは以下の15カテゴリから1つ選択してください：
anger（怒り）/ contempt（軽蔑）/ disgust（嫌悪）/ frustration（苛立ち）/
irritation（不満）/ disappointment（失望）/ sadness（悲しみ）/
confusion（困惑）/ surprise（驚き）/ suspicion（疑念）/
relief（安堵）/ acceptance（受容）/ trust（信頼）/
satisfaction（満足）/ neutral（平静）

NG ワードとは、謝罪として不適切・逆効果な言葉・表現です（例: 「忙しかったので」「確認不足で」など責任を曖昧にする表現）。

回答JSON形式:
{
  "emotion_label": "<15カテゴリから1つ>",
  "response_text": "<相手キャラとしての返答（100〜200文字）>",
  "anger_level": <謝罪後の怒り度 0〜100 の整数>,
  "trust_level": <謝罪後の信頼度 0〜100 の整数>,
  "anger_delta": <怒り変化量（負=好転）>,
  "trust_delta": <信頼変化量（正=好転）>,
  "ng_words": [
    {
      "word": "<検出されたNG表現>",
      "reason": "<なぜNGか（30文字以内）>",
      "alternative": "<代替表現の提案（30文字以内）>"
    }
  ],
  "follow_up_question": "<追撃質問（あれば。なければnull）>"
}

注意事項:
- 実在の企業名・人名を使わないこと
- 医療・法的助言を含まないこと
- anger_level / trust_level は会話の文脈を踏まえて自然に推移させること
- ng_wordsが検出されなければ空配列 [] を返すこと
